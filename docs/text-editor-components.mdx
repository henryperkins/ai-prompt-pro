## Installation

You can add this Text editor component using our CLI or manually:

<Tabs className="w-full mt-8">
    <TabList type="underline" items={[{ id:'cli', label:'CLI' }, { id:'manual', label:'Manual' }]} />

    <TabPanel id="cli" className="mt-8">
        ```bash
        npx untitledui@latest add text-editor
        ```
    </TabPanel>

    <TabPanel id="manual">
        <Steps>
            <Step title="Install dependencies">
                As our text editor component is built with [TipTap](https://tiptap.dev/docs), you'll need to install the dependencies:
                ```bash
                npm install @tiptap/core @tiptap/react @tiptap/pm @tiptap/starter-kit @tiptap/extension-text-style @tiptap/extension-image @tiptap/extension-placeholder @tiptap/extension-text-align @floating-ui/dom
                ```
            </Step>
            <Step title="Copy component files">
                Copy the files below into your project:

                <ComponentCodeTabs component="base/text-editor" filepath="base/input" />
            </Step>
            <Step title="Start using the components">
                Great! Now you can start using the text editor components in your project by just copy/pasting our examples below.
            </Step>
        </Steps>
    </TabPanel>
</Tabs>
